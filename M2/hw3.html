Working with Images

As you can see these two images appear to be identical to the naked eye. I took this photo originally using my DSLR several years ago, exported it to Lightroom where I edited it, and then exported it to a number of file types for optimization for different mediums. The first image you see here is the original JPG version. In the second image, I further optimized it for the web using TinyPNG.com which is a web service that will automatically optimize your images for you. It is compatible with PNG, JPEG/JPG, or WebP file types. Unfortunately (or fortunately, really) I have super fast internet so the loading time of these images is near-instant to the naked eye. When I use developer tools to see the actual data, it takes the original 221ms to load and the compressed version 203ms to load. Interesting!
